# Netflix Movies Analysis using SQL

## Project Overview
This project analyzes Netflix Movies and TV Shows data using SQL to extract insights about content distribution, ratings, genres, release trends, and country-wise contributions.

## Objectives
- Explore Netflix content library
- Analyze movie and TV show distribution
- Identify popular genres
- Study content ratings
- Examine release year trends
- Generate business insights using SQL

## Tools Used
- SQL
- MySQL / PostgreSQL

## Analysis Performed
- Movies vs TV Shows Count
- Content Rating Analysis
- Genre Distribution
- Release Year Trends
- Country-wise Content Analysis
- Top Directors Analysis
- Content Added Over Time

## SQL Concepts Used
- SELECT Statements
- Aggregate Functions
- GROUP BY
- ORDER BY
- CASE Statements
- Joins
- Subqueries
- CTEs
- Window Functions

## Key Business Questions Answered
1. How many Movies and TV Shows are available on Netflix?
2. Which genres are most common?
3. What are the most popular content ratings?
4. Which countries produce the most Netflix content?
5. How has Netflix content grown over the years?
6. Who are the top directors on Netflix?
